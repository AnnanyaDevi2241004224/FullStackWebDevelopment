The json() method of the Request interface reads the request body and returns it as a promise that resolves with the result of parsing the body text as JSON.
The JSON namespace object contains static methods for parsing values from and converting values to JavaScript Object Notation (JSON).
async makes a function return a Promise
await makes a function wait for a Promise
Example
async function myFunction() {
return "Hello";
}
Is the same as:

function myFunction() {
return Promise.resolve("Hello");
}
