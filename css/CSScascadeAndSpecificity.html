CASCADE ALGORITHM :

for solving conflicts where multiple css rules appear on same element

SPECIFICITY CALCULATION :
  1. universal selector :0
  2. element and pseudo selector :1
  3. class,attribute selector and pseudo classes :10
  4. id selector :100
  5. inline styles :1000
ORDER OF SPECIFICITY:

inline style > id selector > class/attribute selector > elemnt selector > universal selector
